Remarks

Represents a range of integers of a specified data type.

Template Parameters
NumAny numeric type
MathTrait type that implements math operations for the type, e.g. for in, use Loyc.Math.MathI.

TODO: unit tests.

Note: if the low value is the minimum value of an integer type (e.g. int.MinValue of int) then the Enumerator will not work. Also, if the difference between the min and max value is int.MaxValue or more, the Count property will overflow and return an incorrect value.
